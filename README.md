# Kliper.City static demo

Это простая статическая сборка сайта:

- `index.html` - входной HTML-файл
- `css/style.css` - стили
- `js/app.js` - логика приложения
- `assets/` - локальные изображения и логотипы

Можно открыть `index.html` напрямую в браузере или загрузить всю папку на простой статический хостинг.
